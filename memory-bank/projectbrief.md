# Project Brief: MacOS Process Monitor

## Overview
A monitoring tool for MacOS that tracks system processes and logs those consuming excessive CPU or memory resources. The tool aims to help users identify resource-intensive applications that may impact system performance.

## Core Requirements
1. Monitor all running processes on MacOS
2. Track CPU and memory usage for each process
3. Set configurable thresholds for "excessive" resource usage
4. Log processes that exceed thresholds
5. Maintain timestamped logs for later analysis
6. Run with minimal impact on system performance

## Goals
- Provide reliable process monitoring with low overhead
- Create detailed, useful logs for troubleshooting
- Support configurable thresholds and monitoring settings
- Develop a simple, intuitive interface
- Make the tool extensible for future enhancements

## Target Users
- System administrators
- Developers troubleshooting performance issues
- Power users concerned about system resource usage

## Success Criteria
- Accurately identifies resource-intensive processes
- Maintains logs without significant disk usage
- Operates with minimal CPU overhead (<1%)
- Configurable to user preferences
- Stable operation over extended periods
